This app was developed using React Native (Expo) and integrates simulated data from Auckland Transport (AT) and NIWA.

**Prerequisites**

Before you start, ensure you have the following installed:
 - A code editor (e.g., VS Code
 - Github
 - Node.js 18+
 - An Android Emulator

**Installation & Setup**
 Clone the repository:
git clone https://github.com/st18036/amdt-mobile.git
cd amdt-mobile

Install dependencies:
npm install react react-native expo
npm install react-native-maps
npm install firebase
npm install @expo/vector-icons
npm install --save-dev typescript @types/react @types/react-native


Open Android Emulator:
Navigate to Device Manager(Icon under Bell Icon on the right)
Select "Add a new device"
Search Pixel 7 and choose it 
In the API dropdown, choose API 34 and select the system image shown
Then press finish and wait for installation
After installation click the play button on the right of the device name 

Start the Expo development server(ONLY Once the pixel 7 is up and running):
npx expo start

Run the app:
On your code editor terminal select "a" for Android
